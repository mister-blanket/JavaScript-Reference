# math


### `Math.floor()`
Drops floating point down to the nearest integer.
`Math.floor(5.9);` // 5
`Math.floor(6.3553);` // 6


### `Math.random()`
Returns a floating point random number > 0 && < 1.
```JavaScript
Math.random(); // > 0 && < 1
Math.floor(Math.random() * 2); // 0 || 1
Math.floor(Math.random() * 100); // 1 to 100
```
